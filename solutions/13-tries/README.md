These solutions are intended to build intuition for trie-based problems. Each solution includes:
1. **Problem Statement**: Brief overview of the goal.
2. **Python Implementation**: Clean, commented code using standard trie patterns.
3. **Complexity Analysis**: Time and space bounds for the solution.
